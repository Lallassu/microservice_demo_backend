About

Browser based multiplayer game made with phaser and nodejs backend. Background graphics from opengameart.org. Author lallassu (github.com/lallassu). The game is currently in beta and there might be some issues with performance and lagg. It works best in google-chrome while there are some performance issues in FireFox. Not tested with other browsers.
